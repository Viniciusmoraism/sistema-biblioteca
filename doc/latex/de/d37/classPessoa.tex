\doxysubsection{Descrição detalhada}
Classe abstrata que representa uma \mbox{\hyperlink{classPessoa}{Pessoa}}. 

A classe \mbox{\hyperlink{classPessoa}{Pessoa}} é uma classe abstrata que serve como base para as classes derivadas que representam diferentes tipos de usuários do sistema.
