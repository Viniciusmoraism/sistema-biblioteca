\doxysubsection{Descrição detalhada}
Classe que representa uma cópia específica de um livro. É responsável por armazenar as informações dos empréstimos, renovações e etc. \mbox{\hyperlink{classCopia}{Copia}} deve ser utilizada primariamente dentro da classe \mbox{\hyperlink{classLivro}{Livro}}.
